UPM Listmaking App

Tech Stack: VueJS, Django, PostgreSQL, Bootstrap CSS, Bulma CSS

For: University Requirements; On: Visual Studio Code

As the culminating project for our class in Database Systems, it's a listmaking app that's heavily inspired by Trello. The user needs to login to access his/her cards or add/remove/edit them. In each cards he/she will have access to a limited number of boards (depends on bought tier) with each having up to a limited number of tasks. The data also comes from a Django REST API, the database of which is managed through PostgreSQL, and the ERD of which was automatically produced from the code.

My Tech Stack

Vue.js

Javascript Framework

Is the framework/library I'm most used to. It makes dynamic HTML easy with the double mustache convention for reflecting data in HTML, the prepended colons for passing data (through the HTML) as props, the v-model attribute for changing the data through the HTML and vice-versa, and more. It's much simpler than ReactJS but with the same capabilities.

Django

Backend Web Framework

Is a high-level Python web framework mainly for the backend (but able to make up the frontend on its own). I consider it similar to Bootstrap CSS but for the backend, where its structure promotes using components and packages for faster development with a good range of capabilities.

PostgreSQL

Object-Relational Database Management System

Despite purportedly being more complex and with broader applications than MySQL, this was the first database management system I've learned to use and so have become more familiar with.

Bootstrap

CSS Framework

I've first used it because of its popularity and have since come to like it because of how good its components look as is and its boon on development speed and name shortcuts for frequently used CSS styles.

React

Javascript Framework

Was the first Javascript framework I've learned to use, but have since switched to Vue.js for the latter's simplicity and cleanliness. Though I'm no longer as familiar with it, I still have enough experience to use it as needed.

Bulma

CSS Framework

Though I've only used it for its navbar, hero, and toast design, and have since learned not to mix different CSS frameworks together, I still can use the technology as needed.
